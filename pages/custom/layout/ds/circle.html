--- notes: > Masonry's great, but the Layout API has one more trick up its sleeve. With a few couple of lines of JavaScript and a little Googled trigonometry, we can make ourself an expanding circular menu that we can distribute with one Layout worklet and a couple of lines of CSS. And that's it. No external dependencies, works with any framework, infinitely distributable. All the makings of an ideal pattern for a design system. ---
.circle-layout { --item-size: 32; --padding: 32; --angle: 0; display: layout(circle); }
