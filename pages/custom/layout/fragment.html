Fragment
  • To perform layout on a box, layoutNextFragment() is called on `LayoutChild` to produce a fragment
  • Contains inlineSize and blockSize attributes, which are set by the layout algorithm and cannot be changed
  • Can be positioned by setting its inlineOffset and blockOffset attributes, which default to 0
